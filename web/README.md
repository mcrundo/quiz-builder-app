# Quiz Web App

Vue3 web application for taking quizzes.

## Features

- Interactive quiz interface
- Progress persistence with localStorage
- Resume capability if user leaves the page
- Display all questions and answers at completion

## Installation

```bash
# Using mise for Node version management
mise use node@24

# Install dependencies
npm install
```

## Usage

```bash
# Generate quiz data (from project root)
question-parser sample-data/SAMPLE-DOCUMENT.docx -o web/public/quiz.json

# Start development server
npm run dev
```

## Development

```bash
# Run tests
npm run test:unit

# Type checking
npm run type-check

# Lint and format
npm run lint
npm run format

# Build for production
npm run build
```

## Requirements

- Node.js 20.19.0+ or 22.12.0+
- Quiz data file at `public/quiz.json` (generated by the parser CLI)

## Usage

The app expects quiz data at `/quiz.json` in the public directory. Use the parser CLI to generate this file from a DOCX document.

### Quiz Data Format

```json
{
  "version": "1.0",
  "questions": [
    {
      "id": 1,
      "text": "What color is the sky?",
      "choices": [
        { "label": "A", "text": "Blue" },
        { "label": "B", "text": "Purple" },
        { "label": "C", "text": "None of the above" },
        { "label": "D", "text": "It depends" }
      ]
    }
  ]
}
```

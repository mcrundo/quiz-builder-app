# Quiz Builder App

A Python parser for DOCX quiz files with a Vue3 web interface.

[![CI](https://github.com/mcrundo/quiz-builder/actions/workflows/ci.yml/badge.svg)](https://github.com/mcrundo/quiz-builder/actions/workflows/ci.yml)

## Project Structure

- `parser/` - Python package for parsing DOCX quiz files into JSON
- `web/` - Vue3 web application for taking quizzes (coming soon)

## Parser

Command-line tool to parse DOCX quiz files and output structured JSON.

See `parser/README.md` for details.
